# 概要

Radio Controller Launcherは、KX3 Controller、FT-817 Controller、FT-857D / FT-897D Controllerのいずれかを移動可能な選択画面から個別に起動するアプリです。

ランチャー自身はCAT通信、シリアルポート、WebSocket、QSOLogMac連携を行いません。各Controllerは引き続き独立したプログラムとして動作します。

# 起動

1. Radio Controller Launcherを起動します。
2. 「無線機コントローラー」画面が表示されます。
3. タイトルバーをドラッグすると好きな位置へ移動できます。

# Controllerの起動

メニューから使用するControllerを1つ選択します。

- KX3 Controller
- FT-817 Controller
- FT-857D / FT-897D Controller

起動済みのControllerには「起動中」とチェックアイコンが表示されます。起動済み項目を選ぶと、そのControllerを前面へ表示します。「すべて起動」機能はありません。Controllerを起動した後もランチャーは終了せず、引き続き利用できます。

別のControllerが起動中に機種を選択すると、起動中のControllerへ通常終了を要求します。終了を確認してから選択したControllerを起動するため、複数のControllerは同時起動しません。終了できなかった場合は安全のため新しいControllerを起動しません。

# アプリの検索場所

次の順序でControllerを検索します。

1. `/Applications`
2. ユーザーの`Applications`フォルダー
3. `/Users/kazuichishinjo/アプリ開発`以下の各開発フォルダー

見つからない場合はApplicationsフォルダーへControllerを配置してください。

# 終了

メニューの「ランチャーを終了」を選択します。Controllerは別アプリのため、ランチャーを終了しても自動終了しません。各Controllerはそれぞれのアプリメニューから終了してください。
